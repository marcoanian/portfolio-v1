Python Japanes Virtual Keyboard Project

by Marco A.

The Japanese Virtual Keyboard Project was born out of my need to create my own vocabulary
JSON files for my studies. Since I wanted a Q&A tool, I found it frustrating to constantly
switch between keyboards. This Virtual Keyboard allows me to:
- Create nested JSON files while converting Hiragana to Romaji and Latin script.
- Practice Hiragana writing.
- Play an interactive Q&A game for learning Japanese.
