Python Calculator Project

by Marco A.

The Python Calculator Project was one of my first complex projects.
Not because of the setup or the GUI, but rather due to the challenge
of processing and calculating data. Since this project utilizes the eval()
function, I had to implement additional input validation, error handling,
and security mechanisms to prevent potential attacks on the user's system.
